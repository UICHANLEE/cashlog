# Cashlog UI Flow

## Home
- 상단: 월 총 지출, 오늘 로그 요약, `+ 기록 추가`
- 중앙: 월 캘린더. 날짜별 지출 합계와 사진 기록 뱃지를 표시한다.
- 하단/우측: 선택한 날짜의 타임라인과 일기형 로그를 표시한다.

## Add Sheet
1. 사용자가 `+ 기록 추가`를 누른다.
2. 하단 시트에서 `카메라/사진 선택` 또는 `직접 입력`을 고른다.
3. 사진 선택 시 파일 선택 input을 열고, 선택 완료 후 mock 분석 결과를 확인 폼에 채운다.
4. 직접 입력 시 빈 폼을 보여준다.
5. 저장하면 시트가 닫히고 선택 날짜 로그가 갱신된다.

## Review Form
- 공통 필드: 제목, 금액, 카테고리, 메모
- 사진 흐름 전용: 이미지 미리보기, 분석 신뢰도, mock 원문
- 저장 전 사용자가 추천값을 수정할 수 있다.

## Daily Log
- 해당 날짜 지출이 없으면 빈 상태를 보여준다.
- 지출이 있으면 시간순으로 카드 목록을 보여준다.
- `DailyLog`는 항목 제목을 기반으로 하루를 일기처럼 요약한다.
